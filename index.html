About Me

Hello! My name is Jitze Jan Kerkstra.
I am a 2nd year ICT student at NHL Stenden Hogeschool.

From idea to design to deployment, I always put in my 100% in every step of the process, I'm driven to make each project my best and want it to be something to be proud of.
I've always had a fascination for security, reliability and friendly design and I try to reflect these traits in my own work.
When it comes to programming I have a lot of valuable experience in different languages and frameworks and I'm determined to learn more.

But I know actions speak more than words so please check out what my website has to offer.


C#

The first programming language I've dealt with and the one that I probably have the most experience with.
I've used C# for console applications, windows applications (WPF) and web applications (.NET Razor Pages), learned valuable programming strategies from it (e.g. MVC and OOP) and had my first interaction with a Database with it.

But because I want to learn new languages, I have decided to keep my distance from using C# in every project, but I wouldn't hesitate using it if a project would benefit from it.

Projects:

  • Themed ForumPage (WebApplication)

C++

After overcoming the stigma of C++ being a hard language to get used to I had a lot of fun learning it.
I had to use C++ to increase the speed of an application by improving algorithms and introducing multithreading.

I would really like to use C++ in future projects because despite the project being very educational I didn't get the chance to add new features (since it was more about improving existing ones).

Projects:

  • Application Speedup (Algorithms and Multithreading)

Java

I started with Java because I used to host my own Minecraft server to play on with friends, this server supported plugins.
I gave my friends to ability to execute commands but this meant they could also use these commands maliciously (shutting down server, banning and kicking other players, etc.).
Because I already had some experience with programming and couldn't find a plugin that suited my needs I decided to make my own using Java and the Spigot framework.
Since then this project has had some neat features added making it useable in other servers. (as of the date of writing this has the code not been made publicly, but I am planning on making it open source once I find some time inbetween other projects)

Other Projects include a 3D Warehouse Simulator with OOP and Pathfinding and I'm currently making a Drawing program with Design Patterns in mind.

Projects:

  • 3D Warehouse Simulator (Spring Framework)
  • Command Blocker Minecraft Plugin (Spigot Framework)
