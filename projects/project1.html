L ibra man Simulatie

3D Warehouse Simulator

Project Description


For this project we were tasked with making a simulation of an automated Warehouse where robots drive around refilling and emptying trucks, these robots need to work together so communication between them was important.

Requirements

  • The Project had to be made by a group of 2 - 4 people
  • We had to make use of Java and Three.js (Javascript/Typscript)
  • The system had to include a client and server, The server simulates the world which the client can see live
  • The system had to make use of a MVC design pattern
  • The Robots had to be able to find their way through the warehouse using pathfinding implementing a shortest path algoritm

Trucks arrive in at the warehouse and need to bring products with them (our products also carry a category that determines to which shelf they belong to), these products need to be stored in the warehouse.
Once the Truck has been emptied by the Robots it will be refilled by the Robots.

Approach


I decided to team up with a good friend of mine for this project.
We prepared a Google Drive for all our documents and a git repository (Github) with the provided code.
The provided code had already implemented Three.js and offered us a square plain with a moveable camera and a moving square representing a Robot.

We decided it would be best to design the warehouse first, the result can be found in the images tab of this project.
